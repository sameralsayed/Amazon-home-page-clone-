# My Store – Responsive E‑commerce Landing Page

## Features

- ✨ Modern responsive layout using **HTML5**, **CSS3**, **Bootstrap 5**, **jQuery**, and **JavaScript**
- 📱 Fully responsive design compatible with **desktop, tablet, and mobile devices**
- 🧭 Top navigation bar with **search bar, account area, and cart indicator**
- 🎠 Hero **carousel** with animated slides and overlay message
- 🧩 **Category cards** section with grid-style content blocks
- 🛍️ Horizontal **product rows** with hover effects and clickable product cards
- 🧮 Interactive **cart counter** that increments when a product card is clicked
- ⬆️ Smooth **“Back to top”** scrolling behavior
- 🧾 SEO-friendly **meta tags**, **Open Graph** tags, and **JSON-LD structured data**
- 🎨 Custom **SVG logo** (not tied to any existing brand)
- 🧱 Flat project structure (no folders): `index.html`, `styles.css`, `script.js`, `README.md`
- 🧑‍💻 Footer credit: **SAMER SAEID – Made by ♥️** with link to `https://bio.link/samer`
- © Copyright **SAMER SAEID**

## Tags

`#html5` `#css3` `#bootstrap` `#jquery` `#javascript` `#responsive` `#ecommerce` `#seo` `#landing-page`

## How to Run

1. **Clone** this repository.
2. Open `index.html` directly in your browser, or serve it with any static server.
3. Ensure you have an internet connection for the Bootstrap and jQuery CDNs.
